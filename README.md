# stef-emoticon

Sebuah Library Php Untuk Mengubah String menjadi gambar emoticon bisa digunakan di framework codeigniter 


Author: Stefanus Prasetyo Nugroho

kode yang tersedia        

    :) atau *happy*
    ;) atau *eyeblink*
    :D atau *bigsmile*
    =-D atau *laugh1*
    =-) atau *laugh2*
    ({}) atau *bighug*
    \=D/ atau *dance*
    <=-P atau *party*
    =)) atau *roftl*
